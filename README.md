# 项目标题

通过Python脚本实时监听微信消息并接收，调用AI模型Api回复消息并发给用户 需要的依赖：Python解释器，wxauto库,openai库


## API 参考

#### 获取API_KEY

- [@octokatherine](https://platform.deepseek.com/api_keys)

| 参数 | 类型     | 描述                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **必选**. 你的 API key |


## 安装

使用 pip 安装 第三方库

```bash
  pip install wxauto
  pip install openai
  pip install OpenAI
```
